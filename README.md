## :notebook: Getting Started
1. Clone this repository
2. `cd backend`
3. `npm install`
4. `cd ../frontend`
5. `npm install`
6. `cd ../mobile`
7. `yarn`
8. `cd ..`
9. To start backend
   1. `cd backend`
   2. `npm start`
10. To start frontend
    1. `cd frontend`
    2 `npm start`
11. To start mobile
    1. `cd mobile`
    2. `yarn start`
12. Enjoy!
